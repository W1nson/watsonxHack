## Setup

1. Install dependencies
```bash
pip install -r requirements.txt
``` 
2. Setup initialize database
```bash
python3 database/setup.py
``` 
3. Run the application
```bash
python3 main.py
```

4. check the API Documentation at http://localhost:8000/docs
